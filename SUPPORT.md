# Support

## Quick checks
- Ensure Cursor Agent auth is configured (`cursor agent login` or `CURSOR_API_KEY`)
- Confirm the extension is installed and enabled
- Run **Developer: Reload Window** after installing a new VSIX

## Logs and artifacts
- Dev Healer output channel: **View → Output → Dev Healer**
- Fix logs and artifacts are written under `.dev-healer/` in your workspace

## Getting help
Open an issue with:
- Extension version
- What you expected vs what happened
- Relevant log excerpts
