# Changelog

All notable changes to **Cursor Dev Healer** will be documented in this file.

This project loosely follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/) and uses SemVer-ish versioning for extension releases.

## Unreleased

## 0.1.65
- Improve conflict-resolution toast: add buttons for **Open conflicted files**, **Stage resolved**, **Copy commands**, and **Open fix log**.

## 0.1.64
- Add conflict helper commands:
  - `Dev Healer: Open Workspace Apply Conflicts`
  - `Dev Healer: Stage Resolved Workspace Conflicts`

## 0.1.63
- Add `devHealer.fixCherryPickStrategy` (default `stashAndPop`) to apply successful worktree fixes back to your workspace even when dirty.

## 0.1.62
- Add `Dev Healer: Rerun Last Fix (Reuse Saved Prompt)`.
